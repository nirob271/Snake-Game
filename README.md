# 🐍 Snake Game

A simple and interactive **Snake Game** built using **HTML, CSS, and JavaScript**.  
This project was created as a practice project to improve my JavaScript logic and DOM manipulation skills.

---

## 🚀 Live Demo

🔗 https://snake-game-js-1234.netlify.app/

---

## 🎮 Features

- Classic Snake gameplay
- Smooth movement using keyboard controls
- Food generation at random positions
- Snake grows when it eats food
- Game over on wall collision
- Restart / Reset functionality
- Modal-based UI (Start / Game Over screen)

---

## 🕹️ Controls

| Key | Action |
|-----|--------|
| ⬅️ Arrow Left | Move Left |
| ➡️ Arrow Right | Move Right |
| ⬆️ Arrow Up | Move Up |
| ⬇️ Arrow Down | Move Down |

---

## 🛠️ Tech Stack

- HTML5
- CSS3 (Grid & Styling)
- JavaScript (Game Logic & DOM Manipulation)
